# ¿Qué es antasoft?

Antasoft es un sistema en desarrollo diseñado para digitalizar y optimizar los procesos internos de la empresa ADVR-Consultores. Su principal objetivo es reducir significativamente los errores humanos que suelen presentarse durante el seguimiento y la gestión de licitaciones para los distintos clientes de la empresa.

El sistema está construido bajo una arquitectura cliente-servidor. En el frontend se emplea una aplicación de página única (Single Page Application, SPA), mientras que en el backend se utiliza una API RESTful encargada de gestionar la lógica del negocio y el almacenamiento de datos en bases de datos estructuradas.
